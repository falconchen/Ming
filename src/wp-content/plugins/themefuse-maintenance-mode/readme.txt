== Description ==

ThemeFuse Maintenance Mode is a coming soon plugin that enables you to 
inform your users when your new website is going to be live while you 
are working on building it. 

<a target="_blank" href="http://themefuse.com/plugin/">View live demo now!</a>

Key Features:

- Twitter integration 
- Newsletter subcribe form
- Time count down 
- Custom admin message
- Custom background 
- Custom logo

Don't forget to check our awesome <a href="http://themefuse.com">Premium Wordpress Themes</a>

== Installation ==

1. Unpack the download-package
2. Upload all files to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Click the settings-link for change time, twitter, text and logo
